# VDT Skill Registry

This registry maps common VDT generation requests to local decomposition skills. It is intentionally small and deterministic for the first agent runtime slice.

| Skill ID | Path | Domain | Matching terms | Primary KPI patterns | Input requirements | Expected outputs | Confidence hints | When not to use |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| mining.production_volume | mining/production-volume.md | mining | production volume, ore mined, throughput, tonnes, availability, utilization, bottleneck | ore mined, ore loaded, production tonnes, plant throughput | calendar_time, downtime, bottleneck_rate, utilization, yield_or_recovery | production_volume, effective_working_time, average_productivity, bottleneck_rate | High when root KPI names ore mined or production volume and context mentions throughput, availability, utilization, or bottlenecks. | Do not use as the primary skill when the request is only about truck cycle productivity with no production volume target. |
