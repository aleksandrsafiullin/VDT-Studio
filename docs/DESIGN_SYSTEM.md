# Design System

VDT Studio uses a calm analytical workspace style:

- Background: true light gray canvas, not cream.
- Surfaces: white panels with thin graphite-blue borders.
- Typography: system sans, compact control labels, strong but restrained headings.
- Accent: muted blue for primary actions and teal for positive calculation signals.
- Status: understated chips for AI, accepted, edited, rejected and issue states.
- Geometry: panels and node cards use 8px or smaller radius unless a control needs a circular icon target.
- Motion: subtle transitions only where state changes benefit from feedback.

The product is an editor, not a marketing page. The first screen must be the usable workspace.
